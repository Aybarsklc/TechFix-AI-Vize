# TechFix AI

**Geliştirici:** Aybars Kılıç
**Amacı:** Saha teknisyeni hata kodu asistanı
**Kullanılan Teknolojiler:** React, Tailwind CSS, Gemini

## Desteklenen Hata Kodları

TechFix AI, sahada sık karşılaşılan hataları anında teşhis edip çözüm adımları sunan akıllı bir asistandır. Anlık olarak tanıyabildiği donanım hata kodları şunlardır:

* **50.4**: Fuser Unit Error
* **13.01**: Paper Jam in Tray 2
* **59.F0**: Transfer Belt Issue
* **79** (veya 79 Service Error): Firmware Corruption
* **005** (veya Banka ATM - 005): Communication Timeout
